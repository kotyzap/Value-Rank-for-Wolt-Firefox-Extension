Privacy Policy for ValueRank for W

Last updated: July 4, 2026 · Firefox build

Short version: ValueRank for W does not collect, store, transmit, or sell any personal data. It has no servers, no analytics, and no remote code. Everything it does happens locally, inside your browser.

What the extension does

ValueRank for W is a Firefox extension that runs only on wolt.com and wolt.cz. On a Wolt category page it reads the price and quantity already displayed on each product card, computes the price per unit (per kg, litre, or piece), and adds a badge and a "Top best value" strip to help you compare products. It does not add items to your cart, submit orders, or modify prices.

Data collection

The extension does not collect any data. Specifically, it does not:

  • Send any information to a server operated by the developer or any third party — the extension has no backend.
  • Use analytics, telemetry, crash reporting, or advertising SDKs.
  • Track your browsing history, purchases, or activity on Wolt or any other site.
  • Collect personal information such as your name, address, email, or payment details.
  • Share, sell, or transfer any data to third parties.

Local storage

The extension uses Firefox's storage API to save a small set of preferences only on your own device:

SettingPurpose
Enabled / disabledWhether sorting and badges are turned on
Best-value countHow many cards appear in the Top strip (Top 4 / 8 / 12 / 16 / 20)
Popup themeLight or dark appearance of the toolbar popup

This data never leaves your device and is not visible to the developer.

Permissions

PermissionWhy it's needed
storageSaves the preferences above locally on your device.
Background script + tab accessLights up the toolbar icon only when you're on a Wolt page, and greys it out elsewhere, by checking the active tab's address. It does not read or collect page content.
Host access to wolt.com / wolt.czLets the content script read the prices and sizes already shown on the page so it can add per-unit badges. It does not run on any other site.

Remote code

The extension does not load or execute any remote code. All JavaScript ships inside the installed package.

Children's privacy

The extension is not directed at children and does not knowingly collect information from anyone, regardless of age, because it collects no information at all.

Changes to this policy

If this policy ever changes — for example, if a future version adds a feature that involves data collection — this page will be updated and the "Last updated" date above will reflect the change.
